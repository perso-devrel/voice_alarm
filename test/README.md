# Test Audio Files

이 폴더는 음성 클론 / TTS 통합 테스트용 오디오 파일을 두는 곳입니다.

- 형식: WAV, MP3, M4A, AAC
- 권장: 30초 이상의 단일 화자 음성 (음성 클론 품질 확보)

## 주의

이 폴더의 모든 오디오 파일은 `.gitignore` 처리되어 **절대 git 에 커밋되지 않습니다**.
사용자가 로컬에 직접 음성 파일을 두면, Ralph 루프와 통합 테스트가 이 파일들을 사용해서 실제 API 검증을 합니다.

## 사용처

- `packages/backend/test/` 의 통합 테스트가 이 폴더의 파일을 읽어 ElevenLabs/Perso 호출
- 수동 검증 시에도 이 폴더의 파일을 사용
