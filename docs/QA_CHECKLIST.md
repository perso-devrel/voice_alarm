# QA 체크리스트

수동 검증용 체크리스트. 주요 사용자 시나리오를 기능 영역별로 정리.

---

## 인증

- [ ] 이메일+비밀번호 가입 → 로그인 → 프로필 조회
- [ ] 잘못된 비밀번호 로그인 시 에러 메시지
- [ ] JWT 만료 후 자동 로그아웃 (401 처리)
- [ ] 웹/모바일 동시 세션

## 음성 관리

- [ ] 오디오 파일 업로드 (10MB 이하, audio/* MIME)
- [ ] 화자 분리(mock) → 1~3명 감지
- [ ] 화자 라벨 편집 (1~50자)
- [ ] 음성 프로필 목록 조회 + 이름 변경
- [ ] 음성 프로필 삭제

## TTS / 메시지

- [ ] 프리셋 메시지 목록 조회
- [ ] 텍스트 입력 → TTS 생성 (mock)
- [ ] 생성된 메시지 재생
- [ ] 메시지 보관함 조회 + 즐겨찾기

## 알람

- [ ] 알람 생성 (시간 + 메시지 + 반복요일)
- [ ] 알람 편집 (시간/반복/활성 토글)
- [ ] 알람 삭제
- [ ] sound-only 모드 설정 시 음성 프로필 필수 검증
- [ ] 알람 목록에서 모드 뱃지 (🔊/🗣️) 확인

## 이용권 / 결제

- [ ] 스텁 결제 → 구독 생성
- [ ] 이용권 코드 발급 (VA-XXXX-XXXX-XXXX 포맷)
- [ ] 이용권 코드 등록 → 구독 활성화
- [ ] 이미 사용된 코드 등록 시 409
- [ ] 발급자에게만 코드 평문 노출

## 가족 플랜

- [ ] 가족 플랜 결제 → 그룹 자동 생성 + owner
- [ ] 초대 코드 발급 (6자리 숫자, 10분 만료)
- [ ] 초대 코드 수락 → 멤버 가입
- [ ] 정원(6명) 초과 시 거부
- [ ] 멤버 탈퇴 / 소유자 권한 양도
- [ ] 멤버 강제 제거 (owner만)

## 가족 알람

- [ ] `allow_family_alarms` 토글 (기본 off)
- [ ] 텍스트 가족 알람 전송 → 수신자 확인
- [ ] 음성 가족 알람 전송 (mock 더빙)
- [ ] 수신자 허용 안 한 경우 403
- [ ] "○○이 보낸 알람" 뱃지 표시

## 캐릭터 / 게이미피케이션

- [ ] 캐릭터 조회 (seed 기본값)
- [ ] XP 지급 → 경험치 증가 + 레벨업
- [ ] 일일 XP 캡(200) 도달 시 capped 응답
- [ ] 성장 단계 전환 시 애니메이션
- [ ] 탭하면 대사 랜덤 전환

## 접근성

- [ ] 스크린 리더로 주요 화면 탐색 가능
- [ ] 모든 버튼에 aria-label 또는 텍스트 존재
- [ ] 터치 타겟 44px 이상 (모바일)
- [ ] 키보드만으로 웹 탐색 가능

## 에러 처리

- [ ] 네트워크 오프라인 시 오프라인 배너 표시
- [ ] ErrorBoundary 동작 (웹/모바일)
- [ ] 잘못된 요청 시 적절한 에러 메시지

## 온보딩

- [ ] 첫 로그인 시 온보딩 오버레이 표시 (웹)
- [ ] 건너뛰기 → localStorage에 완료 저장
- [ ] 3-스텝 끝까지 → 시작하기 → 닫힘
- [ ] 모바일 첫 실행 시 온보딩 화면
