# P15: 모바일 알람 편집 화면

## 선택 항목
P15 첫 번째: 모바일 알람 상세 편집 화면 (시간/반복/메시지 변경)

## 접근
- 기존 alarm/create.tsx 구조를 기반으로 edit.tsx 신규 생성
- 차이점: id 파라미터로 기존 알람 로드 → 필드 사전 입력, updateAlarm API 사용
- target_user_id 변경은 허용하지 않음 (알람 대상은 생성 시 고정)
- 프리셋 TTS 생성 기능은 편집에서 제외 (기존 메시지만 선택 가능)

## 변경 파일
- `apps/mobile/app/alarm/edit.tsx` — 신규 (알람 편집 화면)
- `apps/mobile/app/(tabs)/alarms.tsx` — 알람 카드 onPress → edit 화면 이동
- `apps/mobile/src/i18n/ko.json` — alarmEdit 키 추가
- `apps/mobile/src/i18n/en.json` — alarmEdit 키 추가

## 검증
- `npx tsc --noEmit` 통과 (에러 0)

## 주의사항
- alarms 탭에서 카드 탭으로 편집, 롱프레스로 삭제 (기존 동작 유지)
- 백엔드 PATCH /api/alarm/:id 는 이미 구현되어 있음 (message_id 변경 포함)
