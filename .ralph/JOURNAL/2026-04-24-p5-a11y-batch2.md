# P5: 스택 화면 접근성 Batch 2

## BACKLOG 항목
스택 화면 접근성 Batch 2: voice/record, message/create, dub/translate, family-alarm/create, voice/picker

## 접근

5개 스택 화면에 걸쳐 accessibilityLabel, accessibilityRole, accessibilityState 속성을 일괄 추가했다. 기존 i18n 키를 최대한 활용하되, 스크린 리더 전용 맥락이 필요한 곳은 `a11y*` 접두사 키를 새로 추가했다.

### voice/record.tsx (~8 라벨)
- 가이드 제목: `accessibilityRole="header"`
- 가이드 문장: `accessibilityLabel` (번호 + 문장 텍스트)
- 녹음 버튼: 시작/중지 분기 `accessibilityLabel` + `accessibilityRole="button"`
- 타이머: `accessibilityLabel` + `accessibilityRole="timer"`
- 오디오 레벨: `accessibilityLabel`
- 이름 입력: `accessibilityLabel`
- 제출 버튼: `accessibilityRole="button"` + `accessibilityState: {disabled, busy}`

### message/create.tsx (~20 라벨)
- 섹션 제목: `accessibilityRole="header"`
- 음성 프로필 칩: `accessibilityRole="radio"` + `accessibilityState: {selected}`
- 빈 음성 CTA: `accessibilityRole="button"`
- 탭 (프리셋/커스텀): `accessibilityRole="tab"` + `accessibilityState: {selected}`  + tablist
- 카테고리 칩: `accessibilityRole="radio"` + `accessibilityState: {selected}`
- 프리셋 메시지: `accessibilityRole="radio"` + `accessibilityState: {selected}`
- 커스텀 입력: `accessibilityLabel`
- 생성 버튼: `accessibilityState: {disabled, busy}`
- 미리듣기/정지: `accessibilityRole="button"` + 동적 라벨
- 알람에 사용: `accessibilityRole="button"`
- 선물하기: `accessibilityRole="button"`
- 선물 모달 친구 아이템: `accessibilityLabel` + `accessibilityState: {disabled}`
- 선물 메모 입력: `accessibilityLabel`
- 모달 취소: `accessibilityRole="button"`

### dub/translate.tsx (~12 라벨)
- 메시지 미리보기: `accessibilityLabel` (음성명 + 텍스트)
- 섹션 제목 2개: `accessibilityRole="header"`
- 원본 언어 칩 4개: `accessibilityRole="radio"` + `accessibilityState: {selected, disabled}`
- 대상 언어 아이템: `accessibilityRole="radio"` + `accessibilityState: {selected}`
- 진행 섹션: `accessibilityLiveRegion="polite"`
- 재생 버튼: 동적 라벨 (재생/정지)
- 재시도 버튼: `accessibilityRole="button"`
- 시작 버튼: `accessibilityState: {disabled, busy}`

### family-alarm/create.tsx (~12 라벨)
- 섹션 라벨 4개: `accessibilityRole="header"`
- 수신자 칩: `accessibilityRole="radio"` + `accessibilityState: {selected}`
- 시간 입력: `accessibilityLabel`
- 메시지 입력: `accessibilityLabel`
- 요일 칩 7개: `accessibilityRole="checkbox"` + `accessibilityState: {checked}`
- 제출 버튼: `accessibilityState: {disabled, busy}`
- 빈 상태 이모지 2개: `accessibilityElementsHidden` + 컨테이너에 텍스트 라벨

### voice/picker.tsx (~10 라벨)
- 제목: `accessibilityRole="header"`
- 파일 선택: `accessibilityRole="button"` + 이모지 `accessibilityElementsHidden`
- 업로드 버튼: `accessibilityRole="button"`
- 상태 표시: `accessibilityLiveRegion="polite"`
- 에러 재시도: `accessibilityRole="button"`
- 화자 카드: `accessibilityRole="radio"` + `accessibilityState: {selected}` + 상세 라벨
- 이름 편집 입력: `accessibilityLabel`
- 이름 변경 버튼: `accessibilityRole="button"` + 화자명 포함 라벨
- 닫기 버튼: `accessibilityRole="button"`
