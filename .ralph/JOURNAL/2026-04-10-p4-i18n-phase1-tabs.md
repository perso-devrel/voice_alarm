# P4: i18n Phase 1 — 탭 화면 + 공용 컴포넌트 다국어 적용

## 집은 BACKLOG 항목
- P4: i18n 인프라 (Phase 1: 탭 화면 + 공용 컴포넌트)

## 접근
- 기존 i18n 인프라(i18next, react-i18next, expo-localization)는 이미 설치됨
- 기존 번역 파일에는 home/greeting/screen 키만 있어 홈 화면만 커버
- 전략: (1) 전체 앱 번역 키 작성 → (2) 탭 화면부터 적용 → (3) 다음 루프에서 sub-screen 적용

### 대안
- 모든 화면을 한 번에 적용: 파일 20개 제한에 근접하여 2단계로 분리

## 변경 파일
1. `apps/mobile/src/i18n/ko.json` — 220+ 키로 확장
2. `apps/mobile/src/i18n/en.json` — 동일 구조 영어 번역
3. `apps/mobile/app/(tabs)/_layout.tsx` — 탭 타이틀 i18n
4. `apps/mobile/app/(tabs)/alarms.tsx` — 전체 i18n
5. `apps/mobile/app/(tabs)/voices.tsx` — 전체 i18n
6. `apps/mobile/app/(tabs)/friends.tsx` — 전체 i18n
7. `apps/mobile/app/(tabs)/library.tsx` — 전체 i18n
8. `apps/mobile/app/(tabs)/settings.tsx` — 전체 i18n
9. `apps/mobile/src/components/QueryStateView.tsx` — ErrorView i18n
10. `apps/mobile/src/components/LoginButtons.tsx` — 로그인 버튼 i18n

- Backend `npx tsc --noEmit` — 통과
- Web `npx tsc --noEmit` — 통과
